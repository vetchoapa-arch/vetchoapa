⚙️ Configurar conexión
Para que el correo y Google Meet funcionen de verdad, necesitas publicar tu Google Apps Script como Web App y pegar aquí la URL.
Pasos rápidos:
1. Abre script.google.com y pega el código entregado
2. Clic en Implementar → Nueva implementación
3. Tipo: Aplicación web
4. Ejecutar como: Yo · Acceso: Cualquier persona
5. Copia la URL que aparece y pégala aquí abajo
Esta URL se guarda solo en tu navegador. Nadie más puede verla.
